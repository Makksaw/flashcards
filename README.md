## Installation
1. Clone the repository:
```
git clone https://github.com/Makksaw/flashcards.git
```

2. Navigate to the project directory:
```
cd flashcards
```

4. Install the required dependencies:
```
npm install
```

## Run the project:
```
npm start
```
